# Access Removal Log Sample

This sample illustrates how an access removal event may be documented for audit purposes.

**User:** Jane Smith  
**Reviewer:** Michael Lee  
**Removal Date:** 2024-08-12  
**Reason for Removal:** User changed roles; previous access no longer required.

**Actions Taken:**
- Account disabled.
- Application entitlements removed.
- Shared resource access revoked.
- Remote access permissions removed.

All actions were logged to ensure full traceability and compliance with access governance requirements.
